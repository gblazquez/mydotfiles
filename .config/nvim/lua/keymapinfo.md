# Keymap info

Follow this convention for naming keymaps after <leader>:

- c -> code action
- d -> Display
- f -> Find
- g -> Go
- i -> gIt
- m -> Markdown
- p -> Pretty format
- s -> split, spectre
- t -> tab
- w -> Workspace
- x -> Open trouble diagnostic
- z -> Zen
- z -> Also for spelling (z= and zg)
